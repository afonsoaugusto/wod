# WOD Timer

Contador de tempo para treinos de CrossFit, otimizado para iPad em modo paisagem.

## Uso

1. Configure os blocos **Alongamento**, **Técnica** e **WOD** com exercícios e repetições.
2. Escolha o modo de cada bloco: Sequencial, EMOM, AMRAP ou For Time.
3. Toque em **Iniciar treino** — o relógio grande mostra a contagem regressiva e o painel lateral exibe o exercício atual.
